# Mordomo HA - WhatsApp Bridge Add-on

Ponte directa entre o WhatsApp e o Mordomo HA usando Baileys (mesmo protocolo que o OpenClaw).

## Como funciona

1. Instala o add-on
2. Inicia o add-on
3. Vai ao Dashboard do Mordomo HA → WhatsApp
4. Lê o QR code com o teu WhatsApp
5. Pronto!

## Requisitos

- Mordomo HA instalado via HACS
- Um número de WhatsApp (pode ser o teu pessoal ou um separado)

## Notas

- A sessão do WhatsApp fica guardada entre restarts
- O telemóvel precisa de estar ligado à internet
- Suporta até 4 dispositivos ligados por número (o bridge conta como 1)
